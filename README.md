### テーマを選んだ理由
読書が好きで、ｋｉｎｄｌｅも頻繁に利用しておりますが、
似たようなジャンルに偏ってしまうという悩みがありました。
検索する際にも、どう検索すれば、新しい物に出会えるのか、困っておりました。
検索もシンプルかつ繋げるサービスがあると便利ではないかと考えました。
本が多すぎて、選ぶのに疲れる人の手助けにもなると考え、このテーマにしました。
